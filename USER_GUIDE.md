# 恋爱积分簿 · 使用指南 💕

一本属于两个人的积分簿：记录日常的爱与付出，攒积分换奖励，也记得给对方写一封情书。

网址：**https://yapseng98.github.io**（手机浏览器打开即可，无需安装 App；建议「添加到主屏幕」当 App 用）

---

## 1. 开始使用

### 注册与配对（两个人一次就好）

1. 打开应用，点击开始进入登录页
2. **第一位**（他 💙）：点「注册」→ 选 **他💙** → 填账号和密码 → 注册成功后会得到一个 **6 位配对码**，把它发给对方
3. **第二位**（她 🩷）：点「注册」→ 选 **她🩷** → 填账号和密码 → 输入对方给的配对码 → 注册
4. 配对完成！之后各自用自己的账号密码登录，看到的就是同一本积分簿

> 密码框右边的 👁 可以切换显示/隐藏密码。

### 登录

- 每人用自己的账号登录，登录状态会记住，下次打开不用重新输
- 换设备登录也没问题，数据都在云端

### 本地 Demo 模式

登录页下方有「跳过，使用本地 Demo 模式」——不用账号就能试玩，数据只存在这台手机里。**商店/背包功能在 Demo 模式下不可用**（需要云端）。

从上到下依次是：

## 3. 记分

### 快速记分

1. 在主页确认当前选中的人
2. 「✏️ 快速记分」里选「💙 加分」或「💔 扣分」标签
3. 点一下项目卡片，立刻记上，有小动画和提示

### 自定义记录

点「＋ 自定义记录」可以：选分类、手动改分数、写说明、**改日期**（补记昨天的事就把日期改成昨天）。

### 修改与删除

「📋 本月记录」里每条记录右边：✏️ 编辑（分类、分数、说明、日期都能改）、🗑️ 删除。

### 管理记分项目

「快速记分」旁边的「⚙️ 管理」可以增删改项目（名字、emoji、分数、启用/停用）。正分是加分项，负分是扣分项。

## 4. 每日签到 📅

- 每天签到 **+2 分**，**周日 +5 分**
- 点主页的「每日签到」横幅打开签到页：
  - 日历上每天有两个小点：💙 蓝点=他已签，🩷 粉点=她已签
  - 三个统计：🔥 连续签到天数 · 本月签到天数 · 签到累计积分
  - 大按钮给自己签到；如果对方今天还没签，下面还有一个「💝 帮 TA 签到」按钮
- 每人每天只能签一次

---

## 5. 情书 💌

两个人专属的私密信箱，别的任何人都看不到。

- **写信**：主页点「💌 情书」→ 右上角「✏️ 写信」→ 写内容 → 「封好，送出 💌」
- **收信**：对方的信在你的列表里是一个**密封的信封**（虚线边框，写着「一封悄悄话，点击拆开 💌」）——点开会播放拆信动画，信纸展开才能读到内容
- 拆开后，写信的人那边也能看到信已被读（📖 图标）
- 自己写的信随时可以重新点开看；只有写信人能删除自己的信（🗑）
- 列表里两个人的信混在一起按时间排，卡片上写着是谁写的

## 6. 两种玩法模式

主页的开关随时切换，**两个人共用同一个模式**：

### 🏆 奖励模式（默认）

- 攒**净分数**（加分−扣分−商店消费）冲奖励目标（默认 100 分，可在设置改）
- 「积分表」里从低到高排列各档奖励（如 ≥30 小零食、≥100 想要的东西）
- 月末结算时，各自按达到的最高档拿奖励

### 😈 惩罚模式

- 只看**累计扣分**（做错事扣的分；商店消费不算）
- 扣分堆到惩罚线（默认 80 分，可在设置改）就触发惩罚
- 「积分表」显示惩罚档位（如 ≥30 做一周家务）

> 积分表和惩罚表的档位都能在「积分表」页面右上角管理，或找 ⚙️ 管理按钮增删改。

## 7. 积分表 · 领奖励 · 商店

### 积分表（底部导航第 2 个）

- 奖励模式下显示奖励表：你当前分数够到的档位会亮起「🎁 领取」按钮
- 点「领取」→ 奖励进入你的**背包** 🎒（每个奖励每月只能领一次）
- 惩罚模式下显示惩罚表，当前触发的档位会高亮

### 商店（底部导航第 4 个）🛒

用积分即时兑换小奖励，和月末大奖不冲突：

- **🛒 商店**：两人共同维护的商品架（右上角「⚙️ 管理」加商品，如「奶茶一杯 15 分」）
- 点商品 → 确认兑换 → 扣掉相应积分（会在记录里生成一条「🛒 商店兑换」），东西进背包
- **🎒 背包**：已兑换/已领取还没用的东西；想兑现的时候点「使用」
- **📜 历史**：用过的东西的存档，有购买日期和使用日期
- 背包是**各自的**：谁买的/领的就在谁的背包里
- 分数不够时买不了（按钮会提示）

## 8. 月末结算 🎯

每月底（或想清零重来的时候）：

1. 点主页悬浮的「🎯 月末结算」
2. 预览两个人各自的分数、完成度和结果（奖励模式=各自拿到哪档奖励；惩罚模式=是否触发惩罚）
3. 确认后：本月记录归档，**分数清零**，新的一个月开始
4. 任意一方结算即可，另一方会自动同步

结算过的月份在「历史」（底部导航第 3 个）里可以随时回看：每月两人各自的分数和结果。

## 9. 设置 ⚙️

底部导航最右边（或右上角齿轮）：

| 设置项 | 说明 |
|---|---|
| 奖励目标 | 奖励模式的目标分（默认 100） |
| 惩罚线 | 惩罚模式的触发线（默认 -80） |
| 两人的名字 | 改显示昵称（不影响登录账号）；任意一方改，两边同步 |
| 头像 | 给两只小狗换成自己的照片，也可以一键恢复默认小狗；双方都能改 |
| 在一起开始日期 | 填了之后主页会显示「在一起 xx 天」 |
| 版本号 | 页面底部小字，反馈问题时可以带上 |
| 退出登录 | 换账号用 |

**背景音乐**：右上角 🎵 按钮开关，音量在设置里调。

## 10. 常见问题

**Q：TA 记的分我看不到？**
下拉刷新或点一下底部「主页」，会拉取最新数据。两人是同一本账簿，稍等片刻一定会同步。

**Q：日期显示不对 / 差一天？**
应用按你手机的本地日期记录。如果发现商店购买/使用日期差一天，说明后台脚本是旧版——参考 `servicenow/README.md` 重新粘贴 r28/r30/r32 三个资源脚本即可。

**Q：页面样式乱了 / 功能是旧的？**
强制刷新一次（或清一下浏览器缓存）。应用自带缓存自愈，一般刷新一次就好。

**Q：情书、记录会被别人看到吗？**
不会。所有数据按「配对」隔离，只有你们两个账号能看到；其他任何账号（包括其他情侣）访问都会被服务器直接拒绝。

**Q：手滑记错了怎么办？**
本月记录里直接 ✏️ 改或 🗑️ 删，分数会立刻重算。

**Q：换手机了怎么办？**
新手机浏览器打开网址，用自己的账号密码登录就行，数据都在云端。

**Q：忘记密码？**
目前没有自助找回，需要在 ServiceNow 后台的 `u_love_auth` 表里查看/重置（管理员操作）。
